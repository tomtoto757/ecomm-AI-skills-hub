# 电商 AI 技能库

按电商业务场景整理的 AI 技能集合仓库，方便浏览、复用与持续维护。

## 仓库简介

这个仓库汇总了与电商相关的技能文件，并按照业务类别进行分类，适合用于：

- 查找某个电商场景下可复用的 AI 技能
- 快速浏览不同来源的技能目录
- 作为后续整理、筛选和扩展的基础仓库

## 仓库中包含什么

- `skills/`：按电商业务类别优先分组，再按来源命名空间细分
- `skills/<category>/finsilabs/`：收录并映射自 `finsilabs/awesome-ecommerce-skills` 的相关技能
- `skills/<category>/openclaw/`：收录并映射自 `openclaw/skills` 的电商相关技能
- `README.md`：当前仓库的中文说明文档
- `LICENSE`：仓库许可证文件

## 分类目录

以下为当前仓库的主要业务分类。目录名称保留英文，便于兼容原始结构与链接引用。

| 中文分类 | 目录名 |
|---|---|
| 店铺前台与购物体验 | `storefront-shopping-experience` |
| 商品目录与库存 | `catalog-inventory` |
| 定价、促销与会员体系 | `pricing-promotions-loyalty` |
| 结账、支付与税务 | `checkout-payments-tax` |
| 订单、物流与退货 | `orders-shipping-returns` |
| 客服、支持与 CRM | `customer-support-crm` |
| 营销与增长 | `marketing-growth` |
| 数据分析与报表 | `analytics-reporting` |
| 平台集成与基础设施 | `platform-integrations-infrastructure` |

## 使用方式

1. 打开 `skills/` 目录
2. 进入你关心的业务分类
3. 再按来源查看具体技能
4. 阅读对应技能文件夹中的 `SKILL.md` 与相关评测文件
